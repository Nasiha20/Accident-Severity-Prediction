# Accident-Severity-Prediction
A complete machine learning portfolio to predict the severity of an accident. Several supervised and unsupervised models are discussed.

Python Libraries Used:
- Pandas  1.3.5
- Numpy 1.21.5
- Matplotlib  3.1.3
- Seaborn 0.10.0
- Sklearn 0.24.0
- Yellowbrick 1.3.1
- Kneed 0.7.0
- Fcmeans 1.6.3
- Plotly  5.3.1  
- Pydot 1.4.2
- XGBoost 1.6.0
